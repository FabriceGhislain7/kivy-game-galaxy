# Step 03 - Kivy Basics

## Goal
Understand Kivy app lifecycle.

Key concepts:

- `App` class controls startup and root widget
- `Widget` handles drawing and interaction
- `Clock.schedule_interval` drives the frame loop
- `canvas` draws visual objects (`Rectangle`, `Color`)

Read:

- `src/galaxy_runner/app.py`
- `src/galaxy_runner/game.py`
