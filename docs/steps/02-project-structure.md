# Step 02 - Project Structure

## Goal
Understand where code, assets, and docs live.

- `src/galaxy_runner/` => all runtime code
- `docs/` => architecture + technical guides
- `docs/steps/` => progressive learning path
- `assets/` => game media
- `tests/` => test suite

## Rule
Keep business logic out of ad-hoc scripts and isolate gameplay code in `src/`.
